# Dynamic Event

Comprehensive dynamic event system that creates map-specific NPC encounters with automatic scheduling, configurable spawn positions with radius-based randomization, player count requirements, intelligent spawn validation using hull traces, staff commands for manual event control, and real-time player notifications. Supports multiple concurrent events per map with customizable NPCs, positions, and messages.
