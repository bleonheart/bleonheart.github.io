<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Interactive chess and checkers (draughts) board games with multiplayer support, wagering system, and leaderboard tracking. Features Elo rating system, game history, and comprehensive match management.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Dual Game Modes Both chess and draughts games supported
- Elo Rating System Separate Elo ratings for chess and draughts
- K Factor Dynamic K factor based on games played and rating
- Move System Network messages for synchronized piece movements
- Draw Offers Players can offer and accept draws
- Resignation Players can resign games
- Wagering System Bet money on game outcomes
- Promotion System Pawn promotion selection interface
- Top 10 Leaderboard Database-stored player rankings
- Game Tracking Tracks games played per character
