<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Underground marketplace for stolen goods allowing players to sell items obtained from robberies to fence NPCs. Provides features such as buy stolen items from players, pay based on item reward values, support for all robbery items, and secure transaction system.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Stolen Items Only accepts items marked as stolen from robberies
- Item Pricing Pays based on item reward value range
- Sell Interface Sell stolen items to fence NPC for money
- Buy Interface View available items that can be stolen
- Robbery Integration Works with robbery system items
- Secure Transactions Prevents selling non-stolen items
