<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Bonemerge clothing system that attaches hats, glasses, shirts, pants, shoes, ties, vests, rings, and accessories to players using bone merging. Features vendor purchase system and persistent character customization.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Clothing Slots Hats, glasses, shirts, pants, shoes, ties, vests, rings, neck, wrist, face, and fulloutfit slots
- Bone Attachment Attaches clothing models to player bones for seamless appearance
- Gender Restrictions Male and female specific clothing options with automatic filtering
- Slot Conflicts Cannot wear multiple items in the same slot (prevents overlapping)
- Compatibility System CanWearWith checks for item compatibility between different clothing pieces
- Vendor NPCs Buy clothing from NPC vendors with configurable whitelists and vendor panel UI
- Inventory Management Items removed from inventory when equipped, returned when unequipped
- Character Persistence Equipped clothes saved to character data and persist across sessions
- Auto Spawn Items automatically spawn when unequipped if inventory is full
- Transfer Protection Cannot transfer equipped clothing items to other inventories
- Item Adjustment System Adjust clothing items with position and angle modifications
- Drop Protection Cannot drop equipped items (must unequip first)
