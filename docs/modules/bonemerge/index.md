<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Bonemerge clothing system attaching hats, glasses, shirts, pants, shoes, ties, vests, rings, and accessories to players with vendor purchase and persistence.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Clothing Slots Hats, glasses, shirts, pants, shoes, ties, vests, rings, neck, wrist, face, fulloutfit
- Bone Attachment Attaches clothing models to player bones
- Gender Restrictions Male and female specific clothing
- Slot Conflicts Cannot wear multiple items in same slot
- Compatibility System CanWearWith checks for item compatibility
- Vendor NPCs Buy clothing from NPC vendors with whitelists
- Inventory Removal Items removed from inventory when equipped
- Character Persistence Equipped clothes saved to character data
- Auto Spawn Items spawn when unequipped if inventory full
