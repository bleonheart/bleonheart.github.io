<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Underground contraband trading system allowing players to purchase illegal weapons and contraband through hidden black market dealers. Features limited stock with restock timers, secret locations, configurable pricing, and delayed delivery system for immersive underground roleplay.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Hidden NPCs Black market NPCs teleport between secret locations on configurable timer
- Basket Orders Buy multiple items in a single order with combined pricing
- Import Delay Items arrive after configurable delay timer (default 0 seconds)
- Random Drops Items spawn at random preset locations or player position (configurable)
- Cooldown System Global cooldown between orders and per-item cooldowns prevent spam
- Faction-Based Items Different contraband available per faction
- Waypoint System Automatic waypoint to package drop location when items arrive
- Package Entity Items spawn in physical crate entity that must be opened
- Package Expiration Packages despawn after configurable time limit (default 1800 seconds)
- Proximity Check Must be within 200 units of NPC to place orders
- Inventory Overflow Items automatically drop if player inventory is full
- NPC Configuration Admins can configure import time, despawn time, cooldowns, and location change frequency
