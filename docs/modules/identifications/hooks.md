# ID System Hooks

Hooks provided by the ID System module for identification and documentation systems.

---

Overview

The ID System module provides identification and documentation functionality with various ID types, validation systems, and security features. It includes advanced identity management and verification systems. The module provides extensive hook support for customizing ID validation, documentation processes, security features, and identity management.

---

### OnDescGeneratorCompleted

#### 📋 Purpose
Called when a character description is generated by the description generator.

#### ⏰ When Called
After the description generator completes and produces a final description text.

#### ⚙️ Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `description` | **string** | The generated description text |

#### ↩️ Returns
nil

#### 🌐 Realm
Client
