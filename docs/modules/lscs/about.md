<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Adds compatibility with LSCS (Lightsaber Combat System) for enhanced lightsaber combat mechanics.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- LSCS Integration Compatibility with Lightsaber Combat System addon
- Inventory Items Hilts, crystals, stances, and Force abilities as inventory items
- Dual Wielding Equip hilts and crystals to right or left hand separately
- Hand Occupancy Check Prevents equipping multiple items to same hand
- Saber Crafting Automatically crafts weapon_lscs when items equipped
- Inventory Sync Wipes LSCS inventory on character load
- Visual Indicators Green checkmarks show equipped status per hand
- Item Categories Abilities, Crystals, Hilts, and Stances
