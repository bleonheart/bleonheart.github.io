<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Friendly information and assistance NPCs providing guidance, starter resources, and server information to new players. Provides features such as provides information to new players, can give basic starter items, dialog tree explains server mechanics, directs players to other NPC services, and simple friendly conversation.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Information NPCs Provide guidance and server information to new players
- Starter Items Can give basic items to help new players
- Dialog System Conversational interface explaining server mechanics
- Service Directory Directs players to other NPC services
- Friendly Interface Welcoming experience for new players
