<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Vehicle spawning system that creates physical crates containing vehicles. Players can purchase vehicle crates from NPCs or menus, with configurable vehicle lists and pricing through the AvailableCars table.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Vehicle Crates Spawns physical crate entities containing vehicles
- Crate Items Creates inventory items for each vehicle in AvailableCars table
- Purchase System Buy vehicle crates from NPC or menu interface
- Crate Spawning Spawns crate at player drop position
- Vehicle List Configurable AvailableCars table with prices and descriptions
- Base Item Uses base_vehiclespawner item base for consistent functionality
