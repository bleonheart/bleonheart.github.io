<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Limb damage system where leg shots cause configurable sprint delay with automatic healing and toggleable enable/disable functionality.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Leg Shots Shots to left or right leg disable sprint
- Sprint Disable Configurable delay prevents sprinting
- Auto Heal Leg damage heals automatically after delay
- Config Toggle Enable or disable system via config
- Notifications Player notified when leg damaged and healed
