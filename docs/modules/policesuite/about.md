<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Comprehensive police system with stun weapons, crime tracking, warrants, arrests, police NPCs, and computer database for law enforcement management.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Police Registry Character registry tracks warrants, arrests, fines, help requests, and quiz results
- Police Computer Database system for viewing all records, warrants, fines, prisoners, and help requests
- Police Ranks Five-tier rank system (Recruit, Officer, Sergeant, Lieutenant, Captain) with rank-based salaries
- Rank Management Promote, demote, and set ranks with permission checks per rank
- Rank-based Salaries Different salary amounts per rank level
- Police Locker Entity for accessing rank-based weapon kits once per life
- Weapon Kits Rank-based weapon loadouts (Assault Kit, Patrol Kit) with different weapons per rank
- Warrants System Create, track, and disable warrants in registry
- Wanted Status Shows if player has active warrants
- Arrest System Arrest handcuffed players with configurable duration and crime reasons
- Prisoner Tracking activePrisoners table tracks jailed players with timestamps
- Jailer NPCs NPCs for managing prisoners, viewing active prisoners, and releasing inmates
- Jail Locations Configurable jail cell positions per map
- Criminal Records Comprehensive criminal record tracking in registry
- Arrest History Detailed arrest history with timestamps, officers, and crime reasons
- Fine System Issue and track fines with fine book weapon
- Fine Book Weapon Weapon for issuing fines to players with crime reasons
- Fine Payment Pay fines individually or all at once from bank account or cash
- 911 Command Emergency call system that creates waypoints for all police officers
- Help Requests Track and manage 911 calls in police registry with completion status
- Dispatcher System Manage help requests and emergency calls through police computer
- Request Backup Player interaction menu action for police to request backup with location
- Lockpick Alerts Automatic waypoints for police when doors are lockpicked or lockpick attempts fail
- Nightstick Dual Mode RAG MODE for stunning players or DAMAGE MODE for dealing damage
- Stun Baton Weapon that stuns players with configurable duration
- Serial Numbers Automatic serial number generation for weapons on pickup
- Serial Scratching Scratch serial numbers off weapons using file item
- Weapon Selling Sell weapons to other players with price negotiation
- Weapon Equip System Equip and unequip weapons from inventory
- Police Quiz Quiz system with results tracking and character history
- Quiz Results Track quiz scores, percentages, and grades per character
- Crime Database Extensive crime definitions with codes, sentences, plea deals, and aggravating circumstances
- Emergency Responders Get responders by faction ID
