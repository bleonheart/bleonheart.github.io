<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Comprehensive police system with alarms, 999 emergency calls, stun weapons, crime tracking, warrants, arrests, police NPCs, and computer database for law enforcement management.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Door Alarms Doors trigger alarms when opened
- ECM Jammer Criminals can jam police communications with ecm_jammer item
- Emergency Responders Get responders by faction ID
- Police Registry Character registry tracks warrants and arrests
- Warrants System Create and track warrants in registry
- Wanted Status Shows if player has active warrants
- Arrest System Arrest handcuffed players with duration
- Prisoner Tracking activePrisoners table tracks jailed players
- Jailer NPCs NPCs for managing prisoners
- Police Computer Database system for viewing records
- Stun Baton Weapon that stuns players
- Panic Button 999 emergency call system
