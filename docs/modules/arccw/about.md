<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

ArcCW compatibility module that automatically creates inventory items for all ArcCW weapon attachments, enabling seamless integration between the ArcCW weapon system and the gamemode's inventory system.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Automatic Registration Creates inventory items for all ArcCW attachments from AttachmentTable on module initialization
- Auto Disable Automatically disables module if ArcCW addon is not installed
- Equip System Attachments can be equipped and unequipped directly from inventory
- Transfer Protection Prevents transferring equipped attachments to other inventories
- Auto Equip Equipped attachments automatically re-equip on player loadout
- Hook Removal Removes conflicting ArcCW hooks (PlayerSpawn, PlayerCanPickupWeapon) for compatibility
- Base Item System Uses base_attachment item base for consistent functionality
- Inventory Integration Attachments work seamlessly with the inventory system and ArcCW attachment system
