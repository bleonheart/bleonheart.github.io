<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Welcome cinematic system for new players with 5 scenes (Welcome, Character Creation, World Exploration, Social Interaction, Journey Begins) and smooth camera transitions.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Five Scenes Welcome, Character Creation, World Exploration, Social Interaction, Journey Begins
- Camera Positions Each scene has configurable camera pos, angle, and FOV
- Scene Content Titles, subtitles, and descriptions per scene
- Completion Tracking Tracks if player has seen cinematic
- Admin Skip Superadmins can skip if adminSkip enabled
- Auto Start Plays when character loads if not seen before
- Welcome Messages Shows tips after cinematic completes
- Character Creation Transitions to character creation menu
