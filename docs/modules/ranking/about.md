<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Ranking system for the gamemode with hierarchical faction class progression. Features tier-based ranks, promotion/demotion mechanics, rank-based permissions, salary bonuses, weapon assignments, clearance level integration, and admin management tools.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Tier System Four-tier ranking structure with configurable ranks per class
- Rank Table rankTable per class with tier, salary, weapons, clearance, model
- Promotion Moves player to next tier rank with permission checks
- Demotion Moves player to previous tier rank with validation
- Hire System Set players to tier 1 rank when hired
- Kick System Set players to default class and rank 0
- Rank Permissions Configurable permissions for promote, demote, hire, and kick
- Salary Bonuses Automatic salary from rank data
- Weapon Loadouts Rank-specific weapons granted on spawn
- Clearance Integration Automatic clearance level assignment by rank
- Model Changes Rank can change player model
- Auto Assignment Automatically assigns lowest tier when joining class
