<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Raid-style extraction system requiring players to reach escape zones within countdown timer, with configurable positions per map and audio/visual cues. Features syringe-based extraction mechanics with network messaging.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Extraction Zones Configurable extraction positions per map
- Countdown Timer Players must reach zone within timer limit
- Syringe System Use syringe items to initiate extraction
- Network Messages ReceiveSyringeMessage for extraction communication
- Configurable Time Default 10 second extraction timer
- Map-Based Positions ExtractionPositions table configured per map
- Visual Cues Audio and visual feedback during extraction
