# Configuration

Comprehensive documentation for all marketplace system configurations in the Marketplace NPC submodule.

---

## Overview

This file contains detailed documentation for every configuration setting in the Marketplace NPC submodule. The marketplace system allows players to list items for sale and purchase items from other players.

---

## Basic Settings

### maxListings

- **Description**: Maximum number of items a player can list for sale at one time
- **Type**: Integer
- **Default**: 10

### minimumListingPrice

- **Description**: Minimum price in currency units that can be set when listing an item
- **Type**: Integer
- **Default**: 1

### NPCModel

- **Description**: Model path for the marketplace NPC entity
- **Type**: String
- **Default**: `"models/Humans/Group01/Male_Cheaple.mdl"`
