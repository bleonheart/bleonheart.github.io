<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Player restraint system with ropes/cuffs, blindfolding, gagging, dragging, and body searching mechanics for hostage scenarios.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Handcuff Restraints Prevents weapon use, vehicle entry, door use, and item interaction
- Movement Penalty Handcuffed players move at 50% speed and cannot sprint or jump
- Drag System Drag handcuffed players smoothly with movement sync
- Search System Search inventory of dragged players
- Untie System Remove handcuffs with keys or admin privileges
- Gag System Gagged players cannot use voice chat
- Death Persistence Handcuffs persist through player death
- Bone Animation Handcuffed players show bound arm animations
