<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Player referral system allowing players to refer others and earn monetary rewards when referred players join, with configurable reward amounts and admin management tools.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Referral Selection New players select referrer on character creation
- Database Storage Tracks referred_by, referral_count, last_referral_time
- Configurable Rewards Set reward amounts and cooldown timers
- Max Referrals Configurable limit on referrals per player
- Cooldown System Prevents spam with referral cooldown timer
- Dual Rewards Both referrer and referee get money
- Pending Rewards Rewards saved for offline players
- Admin Management Tools Staff can manage referrals
- Self Referral Check Prevents referring yourself
- Already Referred Check Prevents multiple referrals
