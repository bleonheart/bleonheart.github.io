<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Roleplay animation system with player interaction menu actions for surrender, salute, cross arms, attention, typing poses, and various stances. Animations automatically interrupt when players move, switch weapons, enter vehicles, or sprint, ensuring realistic roleplay interactions.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Animation Actions Player interaction menu actions for various roleplay poses and stances
- Multiple Animations Includes surrender, salute, cross arms, attention, typing, point, halt, and many more
- Bone Manipulation Manipulates player bone angles to create realistic poses
- Movement Detection Automatically interrupts animations on jump, air time, or speed exceeding 450
- Network Synchronization Animations synchronized across all clients via netvars
- Animation State Tracks active animation class and state via netvars
- Auto Interrupt Stops animations on weapon switch, vehicle entry, sprint key press, or movement
- State Restrictions Prevents animations during noclip or other restricted states
- Timed Animations Some animations have automatic offDelay timers (e.g., Timed Salute)
- Bone Reset Automatically resets bone angles when animation ends or player spawns
- Cleanup Handling Resets all animations on map cleanup, item cleanup, and character load
