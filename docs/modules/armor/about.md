<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Armor system with equippable armor items providing damage resistance, speed/jump modifiers, fall damage protection, power armor support, and visual overlays for enhanced character protection and customization.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Damage Reduction Reduces incoming damage based on armor resistance percentage
- Speed Modifiers Adjusts walk speed, run speed, and jump power per armor type
- Fall Protection Can disable fall damage for power armor types
- Power Armor Training Requires PATraining character data to equip power armor
- Screen Overlays Displays overlay images like gas masks and power armor HUD
- Model Changes Changes player model when armor is equipped via group or playerModel
- Bodygroups Skin Changes Bodygroups and skin modifications per armor
- Single Armor Only one armor set can be equipped at a time
- Footstep Sounds Custom sounds for power armor movement
- Radiation Protection Optional RadImmune and RadResistance netvars
