<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Armor system with equippable armor items providing damage resistance, speed and jump modifiers, fall damage protection, power armor support, and visual overlays. Features various armor types including gas masks and power armor variants with themed designs for enhanced character protection and customization.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Damage Reduction Reduces incoming damage based on armor resistance percentage
- Speed Modifiers Adjusts walk speed, run speed, and jump power per armor type (can be positive or negative)
- Fall Protection Can disable fall damage for power armor types
- Power Armor Training Requires PATraining character data to equip power armor
- Screen Overlays Displays overlay images like gas masks and power armor HUD on screen
- Model Changes Changes player model when armor is equipped via group or playerModel, restores on unequip
- Bodygroups Skin Changes Bodygroups and skin modifications per armor with automatic restoration
- Single Armor Only one armor set can be equipped at a time
- Footstep Sounds Custom sounds for power armor movement
- Radiation Protection Optional RadImmune and RadResistance netvars for radiation immunity/resistance
- Equip System Equip and unequip functions with visual indicators in inventory
- Transfer Protection Prevents dropping or transferring equipped armor
- Auto Equip Equipped armor automatically applies stats on player loadout
- Visual Indicators Green indicator when equipped, red when unequipped in inventory
