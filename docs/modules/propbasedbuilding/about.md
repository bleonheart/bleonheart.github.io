<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Constructable prop system with furniture items (boxes, cabinets, chairs, tables, couches) requiring resources and supporting upgrades into functional entities.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Constructable Props Spawn and place building props with resource costs
- Holo Preview Ghost preview of prop before placement
- Placement System Must use keys weapon within 250 units
- Rotation System IN_ATTACK and IN_ATTACK2 rotate prop angles
- Player Collision Check Prevents placing props near other players
- Resource Management Prop placement removes item from inventory
- Prop Upgrades Convert placed props into functional entities
- Health System Optional health values for destructible props
- Persistence Constructable props save across server restarts
- Visual Feedback Color changes show valid placement locations
