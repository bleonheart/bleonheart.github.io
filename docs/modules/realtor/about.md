<h1 style="text-align:center; font-size:2rem; font-weight:bold;">About</h1>

Property creation and management system with tool-based door selection. Allows players to create properties, manage door access, and utilize property storage with network-based data synchronization.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Main Features</h2>

- Property Creation Create properties with tool-based door selection
- Door Selection Select doors to include in property via tool
- Property Database Properties stored in database with name, description, and pricing
- Preview System Preview position and angle for property viewing
- Daily Pricing Configurable price per day for property rental
- Permanent Properties Option to create permanent properties
- Faction Restrictions Properties can be restricted to specific factions
- Door Management Link multiple doors to a single property
- Owner System Track property ownership by SteamID and character
- Network Sync Property data synchronized across clients
